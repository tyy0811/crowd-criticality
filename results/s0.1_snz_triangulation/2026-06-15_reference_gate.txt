=== CONCLUSION ===
The "consistent = Delta-CI contains 0" gate FALSELY REJECTS genuine criticality at large N
(the bias floor; worsens with data; the informativeness ceiling cannot prevent it because
"excludes 0" is not ceiling-governed). The reference-differenced gade -- "consistent =
test-system Delta matches the SAME estimator's Delta on a matched synthetic-critical
reference" -- rescues it: the common estimator bias differences out (critical stays consistent
at all N, d-CI tight around 0) while the look-alike's genuine violation survives (rejected,
d-CI far from 0). Verified for BOTH the converging estimator (curv) and the floored one (tail),
and robust to the x_min choice.

Caveat: fixed x_min narrows the CIs, so the OLD-gate flip appears even for curv here; under
honest reselect-x_min, curv (benign) would likely still contain 0 at this N while tail (floored)
flips. The reference gate rescues both regardless -- that robustness is the point.

=== IMPLICATION (for cross-review, then pre-reg) ===
Gate A.2's null should be the synthetic-critical REFERENCE Delta, not 0. This is the anchor
check for a pre-reg §6 reformulation: "Delta consistent with 0" -> "Delta consistent with the
matched synthetic-critical reference (same estimator, matched N/universality class)". Per the
result-blind discipline the reference is calibrated on the synthetic critical generator only,
never on the real/sim test system's Delta or n. Not locked: cross-review input.
